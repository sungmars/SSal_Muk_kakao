# 🛠️ SSal_Muk – 카카오톡 플레이봇 자동 쌀먹 프로그램

카카오톡 **플레이봇(무기 강화 게임)**의 *쌀먹 자동화* 및 *도전모드(목표 강화 달성)*을 수행하는 프로그램입니다.  
OCR(이미지 문자 인식) 기반으로 동작하며, 카카오톡 입력창에 강화/판매 명령을 자동 입력합니다.

---

## 🚀 1) 모드 선택

두 가지 모드를 제공합니다:

- **쌀먹 모드** — 수익 극대화
- **도전 모드** — 목표 강화 수치까지 자동 강화

## 🎯 2) 목표 강화 수치 입력 (두 모드 동일)

원하는 목표 강화 수치를 입력합니다.

<img width="734" height="473" alt="image" src="https://github.com/user-attachments/assets/920c7480-6c12-4e26-b0d0-689048125aa8" />

## 🖼️ 3) 캡쳐 좌표 설정 (최대 3개)

OCR의 정확도를 위해 **채팅 말풍선이 자주 뜨는 위치를 최대 3곳까지** 드래그하여 선택합니다.  
위치는 순서대로 사용되며, 인식이 애매할 경우 다음 박스로 자동 전환됩니다.

<img width="567" height="1053" alt="image" src="https://github.com/user-attachments/assets/2c631aa9-0099-4a69-ac83-b7ecf6fcbb85" />

---

## ⌨️ 4) 카카오톡 입력창 클릭 → 자동 시작

캡쳐 설정이 끝나면 카카오톡 입력창을 클릭하세요.  
**그 순간부터 강화/판매 명령이 자동 실행됩니다.**

## ⚠️ 주의사항

- 카카오톡 API를 직접 사용할 수 없기 때문에 **화면 캡쳐 방식**을 사용합니다.
- 프로그램 실행 중 카카오톡 위에 다른 창이 올라오면 캡쳐가 실패할 수 있습니다.
- 입력창이 아닌 곳을 클릭하면 정상적으로 입력되지 않을 수 있습니다.
- 본 프로그램은 재미로 만든 프로젝트이며,  
  **사용으로 인해 발생한 손해(골드 손실 등)에 대해 개발자는 책임지지 않습니다.**

---

## 📌 기타 안내

- 강화 성공 시 목표 강화수치 도달 여부를 판정하여 자동 종료됩니다.
- 도전모드에서 목표 강화 도달 시 **사용자 지정 효과음**을 재생합니다.
- OCR 정확도가 낮을 경우 자동으로 흑백 이분화 OCR을 재시도합니다.
- 인식 실패가 연속되면 3개의 캡쳐박스를 순환하며 위치를 변경합니다.
