# SSal_Muk

카카오톡 플레이봇의 쌀먹을 자동화 해주는 프로그램입니다.


1) 캡쳐 좌표 설정1
  <img width="2228" height="1107" alt="image" src="https://github.com/user-attachments/assets/1a367bbc-510d-442d-8bc0-9d816bdd4cd2" />

  왼쪽 상단 모서리의 빨간표시된 부분에 마우스를 올려놓고 "1"을 누릅니다.

3) 캡쳐 좌표 설정2
   <img width="2107" height="1067" alt="image" src="https://github.com/user-attachments/assets/05abf356-43d5-47ee-8702-4748c550dd45" />

   우측 하단 모서리의 빨간표시된 부분에 마우스를 올려놓고 "2"를 누릅니다.

4) 카카오톡 입력창 클릭
   <img width="2110" height="1061" alt="image" src="https://github.com/user-attachments/assets/70d19994-a669-4183-8f41-24f1ff2e77da" />

   2단계까지 진행하셨다면 카카오톡의 입력창을 클릭하면 자동으로 입력이 시작됩니다.


   주의)

   카카오api를 외부로 꺼낼 수 없어 캡쳐방식을 사용중입니다. 프로그램 사용중 카카오톡 위에 다른 창을 올리거나

   다른곳을 클릭하신다면 프로그램 사용에 애로사항이 발생할 수 있으니 주의해주세요.
